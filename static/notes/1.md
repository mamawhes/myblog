# uniapp学习
## view
* 和div差不多
### 属性
* hover-class:指定按下后的样式类
* hover-start-time:进入点击态时间(ms)，默认50ms
* hover-stay-time:松开后保留时间，默认400ms
## text
* 用于包裹文本内容
### 属性
* selectable:文本是否可选，默认为false
## scrill-view
* 可滚动视图组件
### 属性
* scroll-x、scroll-y:允许横向或者纵向滚动
### 如何实现块级元素的横向滚动
* 容器的white-space设置为nowarp

    为了横向排列
* 元素的display设置为inline-block

    设置为行级块元素
